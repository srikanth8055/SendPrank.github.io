# SendPrank

SendPrank is a fun and interactive web application designed to send playful pranks to your friends and family. With an intuitive user interface and easy navigation, it allows users to select various prank options and send them with just a few clicks.

## Features
- Choose from a variety of pranks
- Simple and user-friendly interface
- Share pranks via social media

## How to Use
1. Visit the website.
2. Select a prank from the list.
3. Follow the instructions to send it.
