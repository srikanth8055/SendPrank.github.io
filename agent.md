## Feature Details
- **User Authentication**: Registration and login form with JWT support.
- **Profile Management**: Users can update their information and change passwords.
- **Prank Features**: Users can send pranks on different platforms (details about platforms).
- **Analytics Dashboard**: Track user activity and engagement.

## System Flow
1. **User Registration**:  
   Users create an account using the registration page, which is validated.

2. **User Login**:  
   Registered users can log in to access features.

3. **Profile Setup**:  
   Users complete their profiles after logging in.

4. **Send Prank**:  
   Users choose a prank and execute it by selecting a platform.

5. **Track Analytics**:  
   Users can view their activity through the dashboard.

6. **Logout**:  
   Users must log out to complete the session.
